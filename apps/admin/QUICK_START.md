1. Click "+ Add Portfolio" button
2. Fill in all required fields
3. Click "Create Portfolio"

### Edit Portfolio

2. Modify fields
3. Click "Update Portfolio"

### Delete Portfolio

1. Click "Delete" on any portfolio
2. Confirm in dialog
3. Portfolio removed
